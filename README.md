# Guess the Number Chatbot 🔢

A simple Streamlit chatbot app using OpenAI's GPT-3.5 where users must guess a number from 1-9999.

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://chatbot-dj46w0coyd8.streamlit.app/)

##Overview
After entering an OpenAI API Key, users can begin playing Guess the Number. After each guess, the user will be told if their number is higher or lower than the target number. After guessing the correct number, players have the option to restart the game and start guessing again!

## Instructions

1. Open link to project
2. Login or Sign up to OpenAI
3. Click "Create new secret key" to make a new key
4. Copy your new secret key and enter to start app
5. Read the instructions to play the game
6. Input a guess
7. Check to see if your guess is valid and if it's higher or lower than the number you're guessing
8. Keep going until you guess the correct answer
9. Choose to play again by typing "yes"
